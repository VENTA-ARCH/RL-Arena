# RL Arena Assignments

This repository contains my submissions for the RL Arena program. Throughout these assignments, I explored Python programming, game development using Pygame, and the fundamentals of Reinforcement Learning.

---

## Week 1 – Python Fundamentals

### Objectives
- Learn the basics of Python.
- Become familiar with the development environment and project structure.

### What I Learned
- Python syntax and programming fundamentals.
- Functions, modules, and file organization.
- Translating programming concepts from C++ into Python.

### Outcome
Implemented a simple game while building familiarity with Python and its development workflow.

---

## Week 2 & Week 3 – Territory Capture Game

### Objectives
- Develop a multiplayer grid-based game using Pygame.
- Implement territory capture mechanics inspired by Paper.io.

### Features Implemented
- Two-player gameplay.
- Grid-based movement system.
- Territory ownership and expansion.
- Trail creation while outside territory.
- Territory capture using a flood-fill algorithm.
- Collision detection.
- Score tracking and win conditions.
- Game reset functionality.
- Custom rendering using images and sprites.

### Technical Concepts
- Object-Oriented Programming.
- Game loops and event handling.
- Collision detection.
- Breadth-First Search (BFS).
- Flood-fill algorithms.
- State management.

---

## Week 4 – Reinforcement Learning Introduction

### Objectives
- Learn the fundamentals of Reinforcement Learning.
- Understand Gymnasium environments and PPO training.

### Topics Explored
- Agent-Environment interaction.
- States, actions, and rewards.
- Episode-based learning.
- PPO (Proximal Policy Optimization).
- Gymnasium framework.
- CartPole environment.
- Custom environment design.

### Maze Environment
Studied the implementation of a custom maze environment where an agent learns to navigate from a start position to a goal while avoiding obstacles.
